# Feature Implementation

## Trigger
Event: a feature from `skystream_system_desgn.md` (or a new idea) you want to build.

## Steps
1. **Plan** — re-read the relevant section of `skystream_system_desgn.md`. Identify which layer(s) need to change. Sketch the approach mentally or on paper.
2. **Implement** — write the code.
3. **Test** — run layer-specific tests during iteration, `make test` before commit.
4. **Verify** — `make start`, curl API or check logs.
5. **Commit** — `git commit`.
6. **Push** — `git push`.
